# sprint artifact workspace

this directory is reserved for sprint-level planning, contracts, evaluations, handoffs, and retros.

recommended structure:

`specs/<sprint-id>/plan.md`

`specs/<sprint-id>/contracts/<chunk-id>.md`

`specs/<sprint-id>/evaluations/<chunk-id>.md`

`specs/<sprint-id>/handoffs/<chunk-id>.md`

`specs/<sprint-id>/retro.md`


`specs/<sprint-id>/risen/<chunk-id>.md`

risen prompt contract artifact for dispatch and evaluation consistency.


`specs/examples/`

example payloads for promotion decisions and validation tests.
